# Dog-Breeds
Page for breeds searching

Proyecto individual desarrollado dentro del Henry Bootcamp.

Desarrollo de una SPA utilizando React para el Front End y Redux como state management. Para estilos CSS sin frameworks, Back-end propio en NODE.JS utilizando EXPRESS y Sequelize.js para la BD, utilizando datos de API para perros
Algunas Funcionalidades desarrolladas
- Filtros combinados.
- Barra de busqueda permite buscar razas de perros
- Creacion de raza de perros por form controlados, las cuales se guardan en la BD
- Paginado realizado en JS sin uso de librerías.

Aptitudes: React · Node.js · Sequelize.js · Express.js · PostgreSQL
